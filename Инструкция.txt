Как запустить проект в docker

Необходим установленный Docker Desktop

1. Скопировать проект в папку
2. Открыть консоль в корне проекта
3. Запустить сборку проекта командой: docker-compose up --build
4. При повторном запуске, без сборки можно использовать команду docker-compose up
5. Открыть http://localhost/


Если docker не запускается, можно запустить каждый модуль по отдельности:

1. в папке back выполнить команду yarn dev для запуска сервера
2. в папке front выполнить команду npm run start для запуска клиентской части
3. http://localhost:3000/
